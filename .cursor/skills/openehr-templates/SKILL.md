# openEHR Templates

When helping with openEHR templates (design, review, OET), read and apply the project guides. Use the Read tool to load the relevant file(s) from `resources/guides/templates/`.

Key guides:

- **principles.md** — template design principles
- **rules.md** — normative rules
- **oet-syntax.md**, **oet-idioms-cheatsheet.md** — OET syntax and idioms
- **checklist.md** — conformance checklist

Follow the guidance in the loaded guide(s). Cross-references use `openehr://guides/templates/{name}` (same as MCP).
